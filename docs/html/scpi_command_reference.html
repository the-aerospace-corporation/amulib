SCPI Command Reference

Complete reference guide for all SCPI commands supported by the AMU (Aerospace Measurement Unit)

This document provides a comprehensive reference for all SCPI (Standard Commands for Programmable Instruments) commands supported by the AMU hardware. Commands are organized by functional groups for easy navigation.

Overview

The AMU supports SCPI commands over both USB and I2C interfaces. Commands follow the standard SCPI syntax:

  • Commands are case-insensitive
  • Query commands end with '?'
  • Parameters are separated by spaces or commas
  • Multiple commands can be separated by semicolons

Command Groups

Commands are organized into the following functional groups:

  • System Commands - Device control and status
  • Measurement Commands - Data acquisition
  • DUT Commands - Device Under Test configuration
  • ADC Commands - Analog-to-Digital Converter control
  • Sweep Commands - I-V sweep operations
  • DAC Commands - Digital-to-Analog Converter control
  • Heater Commands - Temperature control
  • Sun Sensor Commands - Solar tracking
  • Memory Commands - EEPROM operations

Command Syntax

Notation

  • = Optional parameters
  • | = Alternative choices
  • # = Numeric suffix (e.g., CH0, CH1, etc.)
  • ? = Query form of command

Examples

*IDN? // Query device identification
SYSTem:LED:COLOR 1.0,0.0,0.0 // Set LED to red
MEASure:ADC:VOLTage? // Measure voltage
ADC:CH0:PGA? // Query PGA setting for channel 0
